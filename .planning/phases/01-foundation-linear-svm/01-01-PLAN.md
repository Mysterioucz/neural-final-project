# 01-01: Data preprocessing and transformation pipeline

## Objectives
- Load the Breast Cancer Wisconsin (Diagnostic) dataset from `data/data.csv`.
- Implement manual label encoding (M -> 1, B -> -1) and feature removal (id).
- Implement a manual `StandardScaler` using only NumPy to ensure feature parity.
- Create a reusable preprocessing pipeline.

## Tasks

<task>
<read_first>data/data.csv</read_first>
<action>
Create `src/data_loader.py` with a function `load_and_clean_data(csv_path)`.
- Use pandas to read the CSV.
- Remove the 'id' and 'Unnamed: 32' (a common artifact in this dataset) columns.
- Return the feature matrix X and target y.
</action>
<acceptance_criteria>
- `src/data_loader.py` exists.

</acceptance_criteria>
</task>
