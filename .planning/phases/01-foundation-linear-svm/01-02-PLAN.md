# 01-02: SVM Dual formulation logic and cvxopt integration

## Objectives
- Implement the core `SVM` class with a `scikit-learn` compatible API.
- Integrate `cvxopt` to solve the dual optimization problem ($P, q, G, h, A, b$).
- Implement a Linear Kernel.
- Support both $w, b$ prediction (efficient) and support vector prediction (extensible).
- Provide basic evaluation metrics.

- `fit(X, y)`:

    - Convert NumPy arrays to `cvxopt.matrix`.
    - Call `cvxopt.solvers.qp`.
    - Extract Lagrange multipliers ($\alpha$).

    - Compute and store $w = \sum \alpha_i y_i x_i$ and $b$ (mean of $y_s - w^T x_s$).
</action>
